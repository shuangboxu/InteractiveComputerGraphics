字符立方体多视角说明（小组成员修改）：

改动概述（小组成员修改）：
在原有字符立方体程序基础上扩展出三种可切换的相机视角，加入玩家模型与对应缓冲区（小组成员修改）：

- 自由飞行相机：保留原有自由移动体验并继续支持滚转与鼠标视角控制（小组成员修改）。
- 第一人称相机：视角固定在玩家头部高度，通过 `1`/数字小键盘 `1` 切换（小组成员修改）。
- 第三人称相机：在玩家身后跟随观察，按 `2`/数字小键盘 `2` 切换（小组成员修改）。
- 再次按 `3`/数字小键盘 `3` 返回自由飞行模式；切换会在窗口标题中展示当前模式名称（小组成员修改）。
- 新增“你”模型数据文件与 GPU 缓冲区，第三人称时会渲染出来（小组成员修改）。

## 操作说明（小组成员修改）
视角控制：

- `1`/`2`/`3`：在第一人称、第三人称、自由飞行相机之间切换（小组成员修改）。
- `W` / `S`：沿前后方向加速或后退（小组成员修改）。
- `A` / `D`：沿左右方向平移（小组成员修改）。
- 鼠标移动：控制视线的左右偏航与上下俯仰（小组成员修改）。
- `Q` / `E`：绕视线方向进行桶滚（小组成员修改）。
- + / -：调整立方体边长 N（控制 FPS）（小组成员修改）。
- [ / ]：调整字间距（小组成员修改）。
- , / .：调整树叶数量上限（小组成员修改）。

程序窗口标题实时显示 N、间距、叶子数和 FPS，并标注当前视角模式（小组成员修改）。

- 焦点切换：当窗口重新获得焦点时，首次鼠标事件用于初始化，避免视角突变（小组成员修改）。

## 性能观察提示（小组成员修改）
飞行或切换视角时可以留意窗口标题栏中实时更新的 FPS，与立方体数量之间的关系用于分析帧率变化（小组成员修改）。
