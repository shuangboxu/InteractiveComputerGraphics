字符立方体飞行相机说明：

改动概述：
本次提交在原有字符立方体程序的基础上，按照“许双博第三次作业修改”的要求，实现了第一人称飞行相机：

- 新增飞行相机状态、辅助函数以及鼠标消息处理，能够实时更新视图与投影矩阵。
- 支持在 `UpdateScene` 中切换到飞行相机逻辑，并根据键鼠输入更新常量缓冲区。
- 保留原有自动取景相机逻辑，可通过 `m_AutoFitCamera` 开关进行控制。

## 操作说明
使用飞行相机时，可通过以下操作控制相机：

 `W` / `S`：沿前后方向加速或后退
 `A` / `D`：沿左右方向平移
 鼠标移动：控制视线的左右偏航与上下俯仰
 `Q` / `E`：绕视线方向进行桶滚
 + / -：调整立方体边长 N（控制 FPS）
 [ / ]：调整字间距
 , / .：调整树叶数量上限
程序窗口标题实时显示 N、间距、树叶数和 FPS。

- 焦点切换：当窗口重新获得焦点时，首次鼠标事件会被用作初始化，避免视角突变。

## 性能观察提示
飞行过程中可以留意窗口标题栏中实时更新的 FPS，与立方体数量之间的关系用于分析帧率变化。
