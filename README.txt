改动概述：
本次提交在原有字符立方体程序的基础上，实现了不同人称飞行相机：

新增飞行相机状态、辅助函数以及鼠标消息处理，能够实时更新视图与投影矩阵。
支持在 `UpdateScene` 中切换到飞行相机逻辑，并根据键鼠输入更新常量缓冲区。
新增数字键快速切换自动取景、第一人称、第三人称和自由飞行四种视角。

操作说明：
使用飞行相机时，可通过以下操作控制相机：

 `W` / `S`：沿前后方向加速或后退
 `A` / `D`：沿左右方向平移
 鼠标移动：控制视线的左右偏航与上下俯仰
 `Space` / `Ctrl`：第一、第三人称模式下升降
 数字键 `1` / `2` / `3` / `0`：在第一人称、第三人称、自由飞行、自动取景视角间切换
 `Q` / `E`：绕视线方向进行桶滚（仅自由飞行模式）
 + / -：调整立方体边长 N（控制 FPS）
 [ / ]：调整字间距
 , / .：调整树叶数量上限
程序窗口标题实时显示 N、间距、树叶数和 FPS。

焦点切换：当窗口重新获得焦点时，首次鼠标事件会被用作初始化，避免视角突变。
角色模型：第三人称与自由飞行模式会显示一个跟随的立方体角色，第一人称下自动隐藏以免遮挡。

性能观察提示：
飞行过程中可以留意窗口标题栏中实时更新的 FPS，与立方体数量之间的关系用于分析帧率变化。
